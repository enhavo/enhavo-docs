Require
-------

.. _route_duplicate:

route
~~~~~

**type**: `string`
**default**: `null`

Define which route should be used to duplicate the selected resource.

.. code-block:: yaml

    actions:
        duplicate:
            type: duplicate
            route: my_duplicate_route
