Download Action
===============

Require
-------

.. _route_download:

route
~~~~~

**type**: `string`
**default**: `null`

Defines which route should be used to download the selected resource.

.. code-block:: yaml

    actions:
        download:
            type: download
            route: my_download_route

Options
-------

.. _ajax_download:

**type**: `boolean`
**default**: `false`

If the value is true, the download request call is executed via "Ajax"-Call in the background.

.. code-block:: yaml

    actions:
        download:
            type: download
            route: my_download_route
